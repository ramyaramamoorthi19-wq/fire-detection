🔥 Fire Detection System

📌 Project Name

Fire Detection using Deep Learning

---

📖 Description

This project is a real-time fire detection system built using deep learning techniques.
It uses a trained Convolutional Neural Network (CNN) model to identify fire from images or video input.
The system helps in early detection of fire hazards and can be used in safety monitoring applications.

---

⚙️ How to Run the Project

1. Clone the repository:
   
   git clone https://github.com/your-username/fire-detection.git

2. Open the project folder:
   
   cd fire-detection

3. Install required libraries:
   
   pip install -r requirements.txt

4. Run the program:
   
   python app.py

---

📊 Dataset Information

- The dataset contains images of fire and non-fire.
- It is used to train the model to classify:
  - Fire
  - Non-fire
- Dataset was collected and preprocessed using Roboflow.

---

🧠 Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- Deep Learning (CNN)

---

🚀 Output

- Detects fire from input image/video
- Displays alert message when fire is detected
